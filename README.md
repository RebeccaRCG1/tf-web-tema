# Galeria Virtual

### Descrição:
Uma pagina web para o compartilhamento de desenhos, obras e artes 
pessoais.

### Principais Funcionalidades
- Exibir imagens das obras feitas pelo artista.
- Exibir informações do artista, contato e outras redes sociais.
